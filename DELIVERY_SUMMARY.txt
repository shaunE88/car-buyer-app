QUICK START (5 MINUTES)
================================================================================

1. Get API Keys
   OpenAI: https://platform.openai.com/api/keys
   RapidAPI: https://rapidapi.com/vin-decoder19-api/api/vin-decoder19

2. Setup
   cd /tmp/car-buyer-app
   cp .env.example .env
   # Edit .env with your API keys

3. Install & Run
   npm install
   npm start

4. Open Browser
   http://localhost:3000

Done! 🎉

================================================================================
API ENDPOINTS
================================================================================

RESEARCH:
  POST   /api/research/by-details       Research by make/model/year
  POST   /api/research/by-vin           Research by VIN
  GET    /api/research/makes            Get car makes
  GET    /api/research/models/:make     Get models
  GET    /api/research/years/:make/:model  Get years

TEST DRIVE:
  GET    /api/test-drive/checklist      Get inspection guide
  POST   /api/test-drive/questions      Get seller questions

NEGOTIATION:
  POST   /api/negotiation/strategy      Get negotiation strategy

HEALTH:
  GET    /api/health                    Server status
